üõ°Ô∏è Moderation

Server moderation tools

ban

Administrator

Ban a user temporarily or permanently

%ban {user} [reason] [time] | /ban
Maximum time: 15 days (s/m/d)

kick

Administrator

Kick the mentioned user

%kick {user} {reason} | /kick

clear

Administrator

Delete the specified number of messages

%clear {number} | /clear
Range: 1-100 messages

warn

Moderator

Warn the mentioned user

%warn {user} {reason} | /warn
Requires prior configuration

unwarn

Moderator

Remove a warning from the user

%unwarn {warnid} {reason} | /unwarn

infracciones

Moderator

View a user's infractions

%infracciones {user} | /infracciones

aislar

Administrator

Temporarily isolate a user

%aislar {user} {time} [reason] | /aislar
Maximum time: 7 days
