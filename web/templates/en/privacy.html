1. Introduction

This Privacy Policy describes how Laslylusky ("the bot", "we") collects, uses and protects information when you use our services on Discord. By using Laslylusky, you accept the practices described in this policy.

2. Information We Collect

We collect the following types of data that are essential for the bot's operation:

2.1 User Data

  • Discord User IDs: To uniquely identify users
  • AI Conversation History: Messages exchanged with the AI assistant to maintain context
  • Level and XP Data: Experience points, current level, message count and last activity
  • Reminders: Titles, descriptions, dates and time zones of configured reminders
  • Birthday Dates: Birth day and month for automatic congratulations
  • Game Statistics: Scores, match history and game statistics (hangman, minesweeper, blackjack)

2.2 Server Data

  • Discord Server IDs: To uniquely identify servers
  • Command Configuration: Commands enabled/disabled per server
  • Permission Configuration: IDs of roles and users with special permissions
  • Log Configuration: Channels and custom messages for audit logs
  • Ticket Configuration: Support ticket system configuration
  • Voice Channel Configuration: Temporary voice channel configuration

2.3 Moderation Data

  • Infractions: Warnings, bans, moderator IDs, reasons and dates
  • Infraction IDs: Unique identifiers for warnings

3. How We Use Information

We use the collected data to:

  • Provide basic bot functionalities (commands, moderation, entertainment)
  • Maintain coherent conversations with the AI assistant
  • Manage leveling systems and rewards
  • Send reminders and birthday congratulations
  • Maintain game statistics and rankings
  • Customize user experience per server
  • Provide moderation functions and audit logs
  • Manage ticket systems and support
