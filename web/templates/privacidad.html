Política de Privacidad de Laslylusky

Última actualización: 26 de mayo de 2025

1. Introducción

Esta Política de Privacidad describe cómo Laslylusky recopila, utiliza y protege la información cuando utilizas nuestros servicios en Discord. Al usar Laslylusky, aceptas las prácticas descritas en esta política.

2. Información que Recopilamos

Recopilamos los siguientes tipos de datos que son esenciales para el funcionamiento del bot:

2.1 Datos de Usuario

  • IDs de Usuario de Discord: Para identificar usuarios únicamente
  • Historial de Conversaciones con IA: Mensajes intercambiados con el asistente de IA para mantener contexto
  • Datos de Nivel y XP: Puntos de experiencia, nivel actual, cantidad de mensajes y última actividad
  • Recordatorios: Títulos, descripciones, fechas y zonas horarias de recordatorios configurados
  • Fechas de Cumpleaños: Día y mes de nacimiento para felicitaciones automáticas
  • Estadísticas de Juegos: Puntuaciones, historial de partidas y estadísticas de juegos (ahorcado, buscaminas, blackjack)

2.2 Datos de Servidor

  • IDs de Servidor de Discord: Para identificar servidores únicamente
  • Configuración de Comandos: Comandos activados/desactivados por servidor
  • Configuración de Permisos: IDs de roles y usuarios con permisos especiales
  • Configuración de Logs: Canales y mensajes personalizados para registros de auditoría
  • Configuración de Tickets: Configuración del sistema de tickets de soporte
  • Configuración de Canales de Voz: Configuración de canales de voz temporales

2.3 Datos de Moderación

  • Infracciones: Advertencias, baneos, IDs de moderadores, razones y fechas
  • IDs de Infracciones: Identificadores únicos para advertencias

3. Cómo Utilizamos la Información

Utilizamos los datos recopilados para:

  • Proporcionar funcionalidades básicas del bot (comandos, moderación, entretenimiento)
  • Mantener conversaciones coherentes con el asistente de IA
  • Gestionar sistemas de niveles y recompensas
  • Enviar recordatorios y felicitaciones de cumpleaños
  • Mantener estadísticas de juegos y rankings
  • Personalizar la experiencia del usuario por servidor
  • Proporcionar funciones de moderación y logs de auditoría
  • Gestionar sistemas de tickets y soporte

4. Acceso a los Datos

El acceso a los datos está estrictamente limitado a:

  • Creador del Bot: vegetines (ID de Discord: 702934069138161835) - Acceso completo para mantenimiento técnico y soporte
  • Usuarios: Solo pueden acceder a sus propios datos de conversación mediante comandos específicos
  • Moderadores de Servidor: Solo pueden acceder a datos de moderación de su propio servidor

5. Almacenamiento y Seguridad

Implementamos medidas de seguridad apropiadas para proteger tus datos:

  • Los datos se almacenan en bases de datos seguras con acceso restringido
  • Se utilizan conexiones cifradas para todas las comunicaciones
  • El acceso está limitado únicamente al personal autorizado
  • Se realizan copias de seguridad regulares para prevenir pérdida de datos

6. Retención de Datos

Los datos se conservan mientras:

  • El bot esté activo en tu servidor
  • Mantengas una cuenta activa en Discord
  • Los datos sean necesarios para proporcionar los servicios
  • No solicites específicamente su eliminación

7. Tus Derechos

Tienes derecho a:

  • Acceso: Descargar tu historial de IA usando el comando /savedatachat
  • Rectificación: Solicitar corrección de datos incorrectos
  • Eliminación: Solicitar la eliminación de tus datos
  • Portabilidad: Obtener una copia de tus datos en formato legible
  • Oposición: Oponerte al procesamiento de tus datos

8. Eliminación de Datos

Para eliminar tus datos:

  • Historial de IA: Usa el comando /reset-chat
  • Otros datos: Contacta a través de un ticket de soporte en nuestro servidor de Discord
  • Servidor de soporte: https://discord.gg/DN6PDKA7gf

9. Compartir Datos con Terceros

No compartimos, vendemos ni transferimos tus datos personales a terceros, excepto:

  • Cuando sea requerido por ley
  • Para proteger nuestros derechos legales
  • Con tu consentimiento explícito

10. Cambios en la Política

Nos reservamos el derecho de actualizar esta política. Los cambios importantes se notificarán a través de:

  • Nuestro servidor de Discord
  • Actualizaciones en esta página web
  • Comandos del bot cuando sea apropiado
