Política de Privacidad de Laslylusky

¿Qué datos se recogen?

En cuanto a los datos que se recogen son los siguientes:

  • Para la IA: se recogen la ID de usuario y el historial de conversación del usuario con la IA
  • Para la configuración: se recoge la ID del servidor, el ID del rol de Mute, ID de roles y usuarios que tengan ciertos permisos exclusivos de comandos del bot en el servidor

¿Para qué son estos datos?

Estos datos son necesarios para el funcionamiento correcto del bot:

  • Para la IA: para que el usuario pueda mantener una conversación seguida con la IA independientemente del canal y del servidor
  • Para la configuración: para una mayor seguridad, configuración y personalización de comandos, permisos y audit logs para el servidor

¿Quién tiene acceso a estos datos?

La única persona que tiene acceso a estos datos es el creador vegelines

Más información

  • Para descargar el historial de IA utiliza el comando %savedatachat. En caso de error contacta con vegelines para solicitar esos datos
  • Para eliminar o restablecer el historial y conversación con la IA utiliza el comando %reset-chat. En caso de error contacta con vegelines
  • Para consultar los datos de configuración del servidor utiliza el comando %config seguido del parámetro data
