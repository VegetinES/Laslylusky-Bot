Descripción General

El sistema de tickets permite a los usuarios crear conversaciones privadas con el equipo de soporte de tu servidor de manera organizada y eficiente. Utilizando hilos privados de Discord, los usuarios pueden abrir tickets para solicitar ayuda, reportar problemas o cualquier otro motivo que hayas configurado.

Proceso de Configuración

Guía paso a paso para configurar el sistema de tickets

1

Crear un Nuevo Ticket

Desde el menú principal, selecciona "Gestionar Tickets" y luego "Crear Nuevo Ticket".

Configuración Inicial:

  • Canal para botones: Donde los usuarios verán los botones de tickets
  • Canal de logs: Para registrar toda la actividad del sistema
2

Configurar Permisos

Define quién puede gestionar o simplemente ver los tickets.

🛡️ Permisos de Gestión

Control total sobre tickets

  • Crear y cerrar tickets
  • Añadir/remover usuarios
  • Archivar tickets
Obligatorio configurar

👀 Permisos de Visualización

Solo ver tickets sin interactuar

  • Ver contenido de tickets
  • Leer conversaciones
  • Sin permisos de gestión
Opcional
3

Personalizar Mensajes

Configura los mensajes que verán los usuarios.

📢 Mensaje para Abrir Tickets

El mensaje principal que contiene los botones

Título personalizable Descripción detallada Color personalizado Imagen opcional

🎫 Mensaje de Ticket Abierto

El mensaje que aparece cuando se crea el ticket

Saludo personalizado Instrucciones claras Información de contacto Tiempo de respuesta
4

Configurar Botones

Añade hasta 5 botones diferentes para distintos tipos de tickets.

🎨 Personalización Visual

  • Texto personalizado
  • Emojis únicos
  • Estilos de botón
  • Colores específicos

🏷️ Formato de Nombres

  • Nombres únicos por tipo
  • Variables personalizables
  • Numeración automática
  • Identificación clara

💬 Mensajes de Respuesta

  • Respuesta automática
  • Contexto específico
  • Instrucciones claras
  • Información relevante
5

Guardar y Desplegar

Al finalizar la configuración, el sistema se activa automáticamente.

🔧

Configuración Automática

Los permisos del canal se configuran automáticamente

📤

Envío de Mensajes

El mensaje con botones se envía al canal seleccionado

✅

Sistema Activo

Los usuarios pueden comenzar a crear tickets inmediatamente

Formato de Nombres de Tickets

Personaliza cómo se nombran los tickets usando variables dinámicas.

{id}
Número único autoincremental por tipo de ticket
Ejemplo: 1, 2, 3...
{userid}
ID del usuario que abre el ticket
Ejemplo: 123456789012345678
{usertag}
Nombre de usuario en Discord
Ejemplo: Usuario#1234

Ejemplos de Formatos

soporte-{id}-{usertag}
soporte-1-Usuario#1234
ticket-{id}
ticket-5
{usertag}-ayuda
Usuario#1234-ayuda
consulta-{userid}
consulta-123456789012345678
